# Feature List


| Feature | Description | Completed? |
|----------|--------------|------------|
| Log in | The user enters a username and password. The system checks credentials and either logs the user in or not. | [x] |
| Log out | The user logs out of the system. | [x] |
| Search blog | The user searches a blog by code (ID). If successful, it returns the blog. | [x] |
| Create new blog | The user creates a new blog with ID, name, URL, and contact email. | [x] |
| Retrieve existing blogs | The user searches blogs by name and retrieves a list containing the search query | [x] |
| Update existing blog | The user searches for a blog by ID, retrieves it, and updates any data. | [x] |
| Delete existing blog | The user deletes a blog by ID. | [x] |
| List all blogs | The user recovers a list of all blogs. | [x] |
| Choose current blog | The user sets a current blog by providing a valid ID. The blog becomes the active working blog until unset. | [x] |
| Create new post in current blog | The user creates a new post for the current blog. The system auto-increments the post ID, stores details, and timestamps creation. | [x] |
| Retrieve existing posts from current blog | The user searches current blog posts by text and retrieves a list of posts that contain the searched text in their title or content. | [x] |
| Update existing post in current blog | The user selects a post by ID, retrieves its details, updates them, and updates timestamps. | [x] |
| Delete existing post in current blog | The user deletes a post by ID in the current blog. | [x] |
| List all posts from current blog | The user lists all posts in a blog from the most recent to the first created post. | [x] |
